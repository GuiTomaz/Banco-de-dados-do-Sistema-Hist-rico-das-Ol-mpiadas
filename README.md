Este repositório contém o projeto final desenvolvido para a disciplina de Banco de Dados I do curso de Ciência da Computação na UFF. O objetivo principal foi aplicar conceitos de modelagem conceitual e lógica para estruturar um sistema capaz de armazenar e gerenciar dados históricos de competições olímpicas.

## Tecnologias e Ferramentas
**SGBD:** MySQL

**Linguagem:** SQL (DDL e DML)

**Modelagem:** Diagrama Entidade-Relacionamento (DER) e Modelo Relacional

## O que foi implementado
O foco do trabalho foi garantir a integridade dos dados e a eficiência nas consultas. Para isso, estruturamos:

**Modelagem Completa:** Gerenciamento de entidades como Atletas, Países (NOCs), Eventos, Equipes e Resultados.

**Integridade Referencial:** Uso rigoroso de chaves primárias e estrangeiras para manter a consistência entre as tabelas.

**Relacionamentos Complexos:** Implementação de cardinalidades e restrições (constraints) que refletem as regras reais do Comitê Olímpico.


Projeto desenvolvido em grupo como parte prática da grade de BD1, cobrindo desde o levantamento de requisitos e desenho do esquema até a implementação dos scripts SQL.
